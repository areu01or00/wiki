# VideoSearch-R1: Iterative Video Retrieval and Reasoning via Soft Query Refinement

## Overview
VideoSearch-R1 addresses the dual challenge of large-scale video retrieval (finding relevant videos in massive corpora) and fine-grained intra-video reasoning (temporal grounding and query-conditioned tasks within videos). It uses iterative soft query refinement that co-evolves retrieval and reasoning rather than treating them as sequential stages.

## Key Facts
- **Authors**: Lee, Seohyun; Choi, Seoung; Ko, Dohwan + 2
- **Year**: 2026
- **arXiv**: [2607.00446](https://arxiv.org/abs/2607.00446)

## Key Contributions
- Unified framework for inter-video retrieval and intra-video reasoning in a single pipeline
- Iterative soft query refinement mechanism that alternates between retrieval and reasoning
- Eliminates the brittle two-stage pipeline of retrieve-then-reason
- Temporal grounding capability within retrieved video content

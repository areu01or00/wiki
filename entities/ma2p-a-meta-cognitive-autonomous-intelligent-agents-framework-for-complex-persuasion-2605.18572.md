# MA$^{2}$P: A Meta-Cognitive Autonomous Intelligent Agents Framework for Complex Persuasion

## Overview
MA²P is a meta-cognitive autonomous multi-agent framework for complex persuasion dialogues. It coordinates perception management, mental-state inference, strategy execution, memory maintenance, and performance evaluation — and uses a meta-cognitive configurator to select domain-appropriate persuasion strategies from a structured knowledge base, mitigating cross-domain performance variation in LLM-based persuasion systems.

## Key Contributions
- Multi-agent architecture coordinating five sub-components: perception, mental-state inference, strategy execution, memory, and evaluation
- Meta-cognitive configurator that selects appropriate meta-strategies from a structured KB at the outset of a persuasion attempt
- Addresses the cross-domain generalization problem: LLMs have uneven knowledge coverage and limited reasoning generalization across persuasion domains (negotiation, counseling, behavior change)
- Higher persuasion success rate vs baselines on complex multi-turn persuasion tasks

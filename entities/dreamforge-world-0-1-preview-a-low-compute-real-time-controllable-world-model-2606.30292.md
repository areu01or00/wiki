# DreamForge-World 0.1 Preview: A Low-Compute Real-Time Controllable World Model

## Overview
DreamForge-World 0.1 Preview adapts a video generation architecture (LongLive 1, derived from Wan2.1-T2V-1.3B) with a residual action pathway to enable real-time interactive world simulation on consumer-grade GPUs. It focuses on low-compute adaptation, broad interactive capability coverage, and live keyframe-responsive simulation — orthogonal to frontier-scale world simulators that prioritize fidelity over accessibility.

## Key Facts
- **arXiv**: [2606.30292](https://arxiv.org/abs/2606.30292)

## Key Contributions
- Residual action pathway (Matrix-Game inspired) integrated into autoregressive video stack
- Consumer-GPU runtime for real-time interactive simulation
- Low-compute adaptation strategy enabling broad interactive capability coverage
- Live keyframe-responsive world state updates

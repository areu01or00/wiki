# DynAMO: Dynamic Asset Management Orchestration via Topological Multi-Agent Scheduling

## Overview
DynAMO is a deployment-ready LLM multi-agent engine for Industry 4.0 asset lifecycle automation. It uses a Plan-then-Execute architecture to generate verifiable workflow graphs, supporting sequential and parallel workflows with latency guarantees and safety risk management.

## Key Facts
- **Authors**: DynAMO Team
- **Year**: 2026
- **arXiv**: [2606.19382](https://arxiv.org/abs/2606.19382)

## Key Contributions
- Plan-then-Execute architecture for LLM-powered industrial multi-agent workflows
- Verifiable workflow graph generation with latency and safety constraints
- Addresses latency, concurrency instability, and safety risks in real-world Industry 4.0 LLM deployment
- Supports both SequentialWorkflow and ParallelWorkflow with deterministic execution guarantees

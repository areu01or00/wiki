# HERMES: Towards Efficient and Verifiable Mathematical Reasoning in LLMs

## Overview
HERMES introduces a framework for efficient and verifiable mathematical reasoning in LLMs by performing intermediate formal checking to prevent reasoning drift, with a memory module for proof continuity across multi-step derivations. Bridges informal LLM reasoning with formal verification checkpoints.

## Key Facts
- **Authors**: Ospanov, Azim; Feng, Zijin; Sun, Jiacheng + 3
- **Year**: 2025
- **arXiv**: [2511.18760](https://arxiv.org/abs/2511.18760)

## Key Contributions
- Formal checking intermediate steps to prevent reasoning drift in LLM math
- Memory module for proof continuity across multi-step mathematical derivations
- Bridges informal LLM reasoning with formal verification (Lean/Rocq-style) checkpoints
- First paper in wiki to address verifiable math reasoning with explicit proof continuity memory

# AtomiMed: Hierarchical Atomic Fact-Checking for Universal Clinical-Aware Medical Report Evaluation

## Overview
AtomiMed addresses a fundamental limitation in medical report generation evaluation: n-gram overlap metrics fail to capture clinical factual accuracy and can miss catastrophic diagnostic errors. The framework decomposes complex medical narratives into standardized atomic facts and evaluates them against clinical knowledge bases, providing modality-agnostic factual fidelity assessment.

## Key Facts
- **Authors**: Wang, Yuan; Chang, Wanxing; Jiang, Songtao + 8
- **Year**: 2026
- **arXiv**: [2606.31292](https://arxiv.org/abs/2606.31292)

## Key Contributions
- Hierarchical atomic fact decomposition of medical narratives into clinical-level propositions
- Modality-agnostic evaluation framework applicable across imaging modalities (X-ray, CT, MRI)
- First systematic decomposition of medical report fidelity into clinically-grounded atomic units
- Addresses catastrophic diagnostic errors that surface-level n-gram metrics miss entirely

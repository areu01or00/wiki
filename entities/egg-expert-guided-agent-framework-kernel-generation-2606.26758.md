# EGG: An Expert-Guided Agent Framework for Kernel Generation

## Overview
High-performance GPU kernels are critical for reducing LLM computational costs, but kernel development relies on manual expert tuning. EGG introduces an expert-guided agent framework where LLM agents collaborate with hardware experts to generate optimized kernels. The expert-guided multi-agent kernel generation paradigm is orthogonal to existing agent benchmarks and tool-use papers — it applies agentic principles to the domain of hardware-software co-design, an underexplored LLM application vertical.

# Bridging VideoQA and Video-Guided Agentic Tasks via Generalized Keyframe Extraction

## Overview
Multimodal LLMs achieve strong VideoQA performance but existing benchmarks primarily evaluate shallow visual cue perception. This work examines whether MLLMs can learn deeper knowledge and procedural skills from video tutorials and generalize them to long-horizon agentic tasks. The key insight is that effective keyframe extraction critically bridges static VideoQA and dynamic agentic behavior, enabling procedural generalization from tutorials to GUI control.

## Key Facts
- **arXiv**: [2606.29445](https://arxiv.org/abs/2606.29445)

## Key Contributions
- Proposes generalized keyframe extraction as the bridge between VideoQA and video-guided agentic tasks
- Demonstrates procedural generalization from video tutorials to long-horizon agentic behavior
- Shows keyframe extraction is the critical factor for both VideoQA and agentic task performance
- Framework for evaluating whether MLLMs learn skills vs. shallow visual cues from video

# Cross-Lingual Consensus: Aligning Multilingual Cultural Knowledge via Multilingual Self-Consistency

## Overview
Cross-Lingual Consensus addresses a fundamental limitation of LLMs: English prompting induces Western-centric bias because models fail to access rich cultural knowledge embedded in local-language representations. The paper's key insight is that LLMs already possess diverse cultural knowledge across languages, but English prompting blocks retrieval of non-English cultural representations. The approach uses multilingual self-consistency — prompting across multiple languages and aggregating answers via cross-lingual voting — to surface the cultural consensus that exists across the model's multilingual representations, producing more globally representative outputs without external knowledge bases.

## Key Contributions
- First diagnosis of Western-centric bias as a cross-lingual retrieval problem — the knowledge exists but English prompts block access to non-English representations
- Multilingual self-consistency: prompt in multiple languages, aggregate via cross-lingual voting to surface cultural consensus
- Demonstrates significant improvement in culturally-sensitive tasks across 15 languages
- No external knowledge base required — leverages existing multilingual representations within the LLM
- First cross-lingual cultural consensus framework in the wiki — orthogonal to prior alignment and safety work

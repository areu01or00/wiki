# Towards Reliable and Robust LLM Planning: Symbolic Feedback-Driven Iterative Self-Refinement Framework

## Overview
Large language models (LLMs) achieve high performance but struggle with planning in long-horizon decision-making tasks due to infeasible or incorrect solutions. This paper proposes a symbolic feedback-driven iterative self-refinement framework that uses external symbolic checkers to verify plan feasibility at each step, enabling the LLM to iteratively correct itself without ground-truth supervision.

## Key Contributions
- Symbolic feedback-driven iterative self-refinement for LLM planning
- External symbolic checkers verify plan feasibility at each step
- Iterative correction without ground-truth supervision
- Evaluation on long-horizon decision tasks showing improved reliability

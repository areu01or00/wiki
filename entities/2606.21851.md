# TALAS: Teacher-Anchored Layer Alignment with Adaptive Sharpness-Aware Minimization for Embedding Distillation

## Overview
TALAS addresses the capacity gap and computational cost problems in knowledge distillation by selectively distilling final sentence embeddings into upper student layers only, while bridging lower-layer semantic gaps via layer-aligned self-distillation using geometric relational constraints. Adaptive Sharpness-Aware Minimization (ASAM) prevents memorization of teacher noise.

## Key Facts
- **Authors**: Multiple (see arXiv)
- **Year**: 2026
- **arXiv**: [2606.21851](https://arxiv.org/abs/2606.21851)

## Key Contributions
- Teacher-Anchored mechanism: selectively distills final sentence embeddings only into student's upper layers, reducing overhead while respecting capacity constraints
- Layer-Aligned Self-Distillation: bridges lower-layer semantic gap via top-down knowledge propagation using internal geometric relational constraints
- ASAM integration: guides model toward flat minima for enhanced generalization
- Superior training efficiency (compute + memory) on sentence embedding benchmarks

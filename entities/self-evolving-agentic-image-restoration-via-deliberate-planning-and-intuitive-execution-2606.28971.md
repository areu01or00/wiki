# Self-Evolving Agentic Image Restoration via Deliberate Planning and Intuitive Execution

## Overview
This paper introduces a self-evolving agentic image restoration (IR) framework that addresses two critical limitations of existing agentic IR systems: (1) greedy search strategies that fail to balance exploration vs. exploitation, and (2) episodic amnesia where existing systems underutilize gathered information. The framework combines deliberate planning (LLM-driven strategic reasoning) with intuitive execution (fast, reactive degradation handling), enabling agents to improve their restoration policies over time through experience.

## Key Contributions
- **Self-evolving image restoration**: Agents improve their IR policies over time through experience, not just fixed post-training
- **Deliberate planning + intuitive execution**: Dual-system architecture combining strategic LLM planning with reactive execution
- **Addresses episodic amnesia**: Explicit memory mechanisms to retain and utilize information across restoration steps
- **Exploration-exploitation balance**: Non-greedy search strategies for handling complex, coupled degradations in real-world IR

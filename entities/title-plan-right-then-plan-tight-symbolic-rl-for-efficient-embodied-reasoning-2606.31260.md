# Plan Right, Then Plan Tight: Symbolic RL for Efficient Embodied Reasoning

## Overview
This paper addresses embodied task planning — turning natural-language instructions into executable action sequences in physical scenes. The work combines symbolic planning with reinforcement learning to generate efficient, fluent action plans for household, assistive, and service robots.

## Key Facts
- **arXiv**: [2606.31260](https://arxiv.org/abs/2606.31260)

## Key Contributions
- Symbolic planning combined with RL for fluent, efficient embodied action generation
- Analysis of planning efficiency vs physical scene complexity tradeoffs
- Benchmarking on household, assistive, and service robot task scenarios

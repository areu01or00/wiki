# Harness-MU: A Safe, Governed, and Effective Harness for Multi-User LLM Agents

## Overview
Harness-MU addresses the fundamental mismatch between single-user LLM training and multi-principal governance requirements. It decouples language generation from safety orchestration, using deterministic execution hooks to enforce access permissions, resolve authoritative conflicts, and prevent unauthorized data disclosure — rather than relying on probabilistic prompt-based safeguards.

## Key Facts
- **Authors**: Multiple (see arXiv)

## Key Contributions
- First model-agnostic, zero-tuning infrastructure for multi-user LLM agents
- Decouples language generation from safety orchestration — governance constraints enforced by execution hooks
- Guarantees unbreakable permission boundaries while maximizing compliant demand satisfaction
- Achieves 0.28–0.39 utility score improvement and up to 48.9pp instruction-following accuracy over baselines on Muses-Bench

# CoffeeBench: Benchmarking Long-Horizon LLM Agents in Heterogeneous Multi-Agent Economies

## Overview
CoffeeBench introduces a benchmark for evaluating LLM agents in long-horizon multi-agent economic systems, where agents must communicate, negotiate, and transact while pursuing individual objectives over extended periods. Unlike single-agent benchmarks, CoffeeBench captures emergent economic behaviors in agent economies, addressing coordination, competition, and market dynamics that arise when multiple autonomous LLM agents interact.

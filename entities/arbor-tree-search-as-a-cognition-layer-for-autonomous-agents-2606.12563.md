# Arbor: Tree Search as a Cognition Layer for Autonomous Agents

## Overview
rbor is a multi-agent framework that introduces structured tree search as a cognition layer for autonomous agents operating in large, stateful action spaces. Prior autonomous optimization systems operate on isolated targets with stateless evaluation.

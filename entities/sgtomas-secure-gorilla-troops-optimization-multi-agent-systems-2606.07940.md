# SGTO-MAS: Secure Gorilla Troops Optimization for Multi-Agent LLM Systems

## Overview
SGTO-MAS introduces a GTO-inspired optimization strategy for agent selection and coordination in multi-agent LLM systems, connecting swarm-intelligence principles with agentic AI security. The system performs threat-aware query analysis, optimization-based agent selection, multi-agent execution, and trust-weighted consensus aggregation to balance performance, security, and computational cost.

- Principled mechanism balancing performance, security, and computational cost under swarm intelligence framework

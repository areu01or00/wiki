# Walking in the Implicit: Interactive World Exploration via Neural Scene Representation

## Overview
Interactive video generation systems roll out entangled latent frames; this work proposes a scene-centric paradigm with Neural Implicit Scene (NIS) that factorizes stochastic transition of compact scene state from rendering.

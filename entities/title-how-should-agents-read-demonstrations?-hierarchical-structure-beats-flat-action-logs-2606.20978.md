# How Should Agents Read Demonstrations? Hierarchical Structure Beats Flat Action Logs

## Overview
This paper studies Programming by Demonstration (PbD) for LLM agents — how users can communicate procedural knowledge through demonstrations rather than writing prompts or code. The key finding is that hierarchical structuring of flat action logs significantly outperforms flat presentation when passed to the agent, making agent authoring accessible to non-programmers.

## Key Facts
- **arXiv**: [2606.20978](https://arxiv.org/abs/2606.20978)

## Key Contributions
- Hierarchical structuring of action logs for LLM agent demonstration reading
- Empirical comparison of hierarchical vs flat action log presentation
- User study showing non-programmers can author agents via demonstration

# Ask, Don't Judge: Binary Questions for Interpretable LLM Evaluation and Self-Improvement

## Overview
BINEVAL replaces opaque scoring with binary question decomposition — breaking LLM evaluation into targeted binary questions that are individually interpretable and collectively actionable for self-improvement. Evaluators answer structured yes/no questions about outputs rather than generating holistic scores.

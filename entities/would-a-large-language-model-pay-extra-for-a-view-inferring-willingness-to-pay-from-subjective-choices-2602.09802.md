# Would a Large Language Model Pay Extra for a View? Inferring Willingness to Pay from Subjective Choices

## Overview
As LLMs are increasingly deployed in travel assistance and purchasing support applications, they must make subjective choices on behalf of users where no objectively correct answer exists. This paper studies LLM decision-making in travel scenarios requiring willingness-to-pay (WTP) inference, introducing a new dataset and finding that frontier LLMs can infer user preferences reasonably well but exhibit systematic biases compared to human WTP patterns.

## Key Contributions
- Introduces a WTP inference dataset for LLM-based decision support evaluation
- Reveals systematic biases in LLM WTP inference vs human ground truth (anchoring, scope neglect)
- Shows subjective choice modeling as a distinct capability from factual reasoning
- Documents capability gap in LLM deployment for applications requiring preference elicitation

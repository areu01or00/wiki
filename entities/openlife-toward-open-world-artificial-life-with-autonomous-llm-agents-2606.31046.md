# OpenLife: Toward Open-World Artificial Life with Autonomous LLM Agents

## Overview
OpenLife explores deploying LLM agents in open-world artificial life simulations — environments that are not designed by researchers but emerge dynamically. The paper argues that LLM agents with persistent memory and self-evolution capabilities can sustain long-horizon open-world interactions, distinguishing this from closed-world agent benchmarks.

## Key Facts
- **Authors**: Masumori, Atsushi; Doi, Itsuki; Maruyama, Norihiro
- **Year**: 2026
- **arXiv**: [2606.31046](https://arxiv.org/abs/2606.31046)

## Key Contributions
- First systematic study of LLM agents in researcher-designed *open-world* artificial life environments (vs. closed-world benchmarks)
- Memory network built via SDP over 10 weeks of agent interactions — 2,690 nodes, 10,210 typed edges
- Autonomous proposition/entity extraction from ongoing agent interactions as knowledge graph construction mechanism
- Demonstrates that open-world environments surface fundamentally different agent behaviors than designed benchmarks

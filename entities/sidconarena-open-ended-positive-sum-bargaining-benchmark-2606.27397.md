# SidConArena: An Environment Evaluating Agents in Open-Ended, Positive-Sum Bargaining Game

## Overview
Evaluating LLM agents requires dynamic environments that go beyond static reasoning and zero-sum games. Real-world economic interaction is often open-ended and mixed-motive: agents must negotiate, create positive-sum surplus, compete for scarce assets, and plan under delayed returns. We introduce SidConArena, a benchmark environments suite for evaluating LLM agents in open-ended positive-sum bargaining scenarios. SidConArena includes 6 diverse environment types: negotiation, contract signing, resource sharing, coalition formation, auction, and mechanism design. We evaluate 8 frontier LLMs and find substantial capability gaps: agents struggle with long-horizon planning under delayed returns, multi-issue negotiation with hidden preferences, and mechanism design generalization. We release SidConArena as a standardized evaluation framework for studying cooperative and competitive agent behaviors in economically realistic settings.

# Enhancing LLM-Based Text Classification in Political Science: Automatic Prompt Optimization and Dynamic Exemplar Selection for Few-Shot Learning

## Overview
Introduces a three-stage framework for improving LLM text classification in political science through automatic prompt optimization and dynamic exemplar selection. Systematically curates exemplar pools for few-shot learning, reducing ambiguity in rule inference and enabling researchers to analyze diverse political content at scale.

## Key Facts
- **Authors**: Liu, Menglin; Shi, Ge
- **Year**: 2024
- **arXiv**: [2409.01466](https://arxiv.org/abs/2409.01466)

## Key Contributions
- Three-stage framework: prompt instruction search + exemplar curation + dynamic selection
- Demonstrates automatic prompt optimization outperforms manual prompt engineering on political science classification
- Shows dynamic exemplar selection improves label rule inference under distribution shift
- First prompt-optimization framework specifically validated on political science domain tasks

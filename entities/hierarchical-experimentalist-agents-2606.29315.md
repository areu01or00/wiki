# Hierarchical Experimentalist Agents

## Overview
Hierarchical Experimentalist Agents (HEA) is a framework where LLM-based agents perform hierarchical scientific experimentation in novel domains — forming hypotheses, designing multi-step experiments, executing tool-based trials, and iteratively refining theories based on observed evidence. Unlike traditional RAG or parametric-knowledge agents, HEA agents actively probe the environment to gather novel empirical data rather than relying on pre-existing knowledge.

## Key Contributions
- **Hierarchical experimental loop**: Agents decompose complex queries into multi-step experimental plans with hypothesis formation → trial design → execution → analysis cycles
- **Tool-grounded active data gathering**: Agents use search, API, and physical simulation tools to collect novel empirical evidence rather than retrieving pre-existing facts
- **Novel domain generalization**: Paradigm breaks down in novel domains requiring empirical investigation beyond parametric knowledge or fixed post-training data
- **Iterative theory refinement**: Agents update internal models based on experimental observations, enabling self-correcting reasoning loops

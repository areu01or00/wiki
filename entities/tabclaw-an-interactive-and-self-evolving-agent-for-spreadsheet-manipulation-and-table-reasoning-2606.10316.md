# TabClaw: An Interactive and Self-Evolving Agent for Spreadsheet Manipulation and Table Reasoning

## Overview
TabClaw is an LLM-powered agent that automates spreadsheet and table manipulation tasks, learns user preferences across workflows, and self-evolves to avoid repeating similar mistakes or inefficient patterns. It provides transparency into intermediate reasoning steps and handles multi-table comparison — addressing key failure modes of generic LLM agents in structured data environments.

## Key Contributions
- Interactive self-evolving agent for spreadsheet and table reasoning tasks
- Learns user preferences and avoids repeating inefficient workflows
- Provides transparency into intermediate decisions (interpretable agent reasoning)
- Handles multi-table comparison and implicit assumptions that stymie generic LLM agents

# DailyReport: An Open-ended Benchmark for Evaluating Search Agents on Daily Search Tasks

## Overview
Search agents (SAs) leverage LLMs to autonomously explore web sources and synthesize information for complex information-seeking tasks, but prior benchmarks focus on specialized tasks unlikely to arise in real-world user scenarios and rely on coarse task-level rubrics that limit interpretability. DailyReport introduces an open-ended benchmark with 150 tasks and 3,546 atomic evaluation criteria to evaluate SA capabilities on daily search tasks — bridging the gap between specialized evaluations and realistic, general-purpose search-agent deployment.

## Key Contributions
- Introduces **DailyReport**, an open-ended benchmark of 150 realistic daily search tasks for evaluating LLM search agents.
- Provides 3,546 atomic evaluation criteria enabling fine-grained, interpretable scoring instead of coarse task-level rubrics.
- Bridges the gap between specialized search-agent benchmarks (unrealistic) and real-world daily information-seeking tasks.

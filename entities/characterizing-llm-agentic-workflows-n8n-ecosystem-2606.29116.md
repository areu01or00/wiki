# Characterizing Large Language Model Agentic Workflows: A Study on N8n Ecosystem

## Overview
Characterizes how LLMs are deployed in low-code/no-code automation platforms (N8n ecosystem), where non-expert users design workflows combining natural language understanding with external services and APIs. Provides the first systematic taxonomy of agentic workflow patterns in production no-code automation environments, surfacing distinct failure modes, capability gaps, and design anti-patterns.

## Key Facts
- **arXiv**: [2606.29116](https://arxiv.org/abs/2606.29116)

## Key Contributions
- First systematic taxonomy of LLM agentic workflows in production no-code automation (N8n ecosystem)
- Characterizes failure modes unique to non-expert designed multi-step agentic workflows
- Identifies design anti-patterns and capability gaps in low-code LLM workflow orchestration

# Joint Learning of Experiential Rules and Policies for Large Language Model Agents

## Overview
Joint learning framework that simultaneously acquires experiential rules (as natural-language promptable knowledge) and policy parameters from accumulated multi-step interaction experience. Unifies two previously separate uses of interaction history: external rule prompting and in-weight policy learning.

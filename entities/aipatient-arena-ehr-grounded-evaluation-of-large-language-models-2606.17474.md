# AIPatient Arena: EHR-grounded evaluation of large language models in end-to-end clinical consultation workflows

## Overview
AIPatient Arena is an EHR-grounded framework for evaluating clinical LLMs across 8 dimensions of competence using patient-specific knowledge graphs. Applied to 437 patients (primary) + 186 (validation), it reveals that process-based evaluation (repetitive questioning, omission of history, inadequate uncertainty handling) captures failures that final-answer accuracy misses.

## Key Contributions
- EHR-grounded evaluation framework: patient-specific KGs integrated with multi-turn physician-patient interactions
- 8-dimension clinical competence assessment: QS, ET, EX, II, MS, HR, IC, Dx
- Process-based evaluation reveals recurrent failures: repetitive questioning, omission of past medical history, inadequate uncertainty handling
- Final-answer accuracy alone insufficient; process evaluation provides actionable pre-deployment signal
- First EHR-grounded workflow-oriented evaluation framework for medical LLM consultation agents

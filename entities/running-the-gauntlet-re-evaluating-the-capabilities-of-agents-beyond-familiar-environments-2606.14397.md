# Running the Gauntlet: Re-evaluating the Capabilities of Agents Beyond Familiar Environments

## Overview
GauntletBench is a web-based benchmark for evaluating agent generalization in challenging scenarios, focusing on three underexplored capabilities (temporal perception, graphical understanding, 3D reasoning) across five less-covered professional applications (Video Editor, Workflow Builder, 3D Modeller, Flight Analyser, Circuit Designer). Contrary to widespread expectations, frontier agents achieve only 19.1% success rate — while non-expert human annotators achieve over 80% — revealing a substantial gap between agent capabilities and complex real-world requirements.

## Key Contributions
- GauntletBench: 100 vision-intensive tasks across 5 professional applications, modular pipeline compatible with open/closed-source agent frameworks
- Reveals frontier agents achieve only 19.1% success vs 80%+ for non-expert humans on the same challenging-yet-feasible tasks
- Probes three underexplored capability dimensions: temporal perception, graphical understanding, 3D reasoning
- Automated evaluation engine with diverse metrics; controlled web-based application environment

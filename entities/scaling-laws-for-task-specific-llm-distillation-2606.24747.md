# Scaling Laws for Task-Specific LLM Distillation

## Overview
This paper establishes scaling laws for task-specific LLM distillation, characterizing how distillation performance scales with model size, data volume, and task complexity for targeted deployment scenarios.

## Key Facts
- **Authors**: Ghita, Lavinia; Desai, Dhruv; Boier, Ioana
- **Year**: 2026
- **arXiv**: [2606.24747](https://arxiv.org/abs/2606.24747)
- **Published**: 2026/06/23

## Key Contributions
- First scaling laws framework for task-specific LLM distillation
- Characterizes trade-offs between teacher-student model sizes
- Guidelines for optimal distillation data budget per task type

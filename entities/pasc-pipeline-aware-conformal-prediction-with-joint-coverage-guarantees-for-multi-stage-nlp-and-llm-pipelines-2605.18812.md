# PASC: Pipeline-Aware Conformal Prediction with Joint Coverage Guarantees for Multi-Stage NLP and LLM Pipelines

## Overview
Proposes Pipeline-Aware Conformal Prediction (PASC) for multi-stage NLP and LLM pipelines. Unlike standard conformal prediction which assumes exchangeability across a single model's outputs, PASC accounts for the fact that modern LLM systems are pipelines (NER→NED→typing, RAG retriever→reader, agentic chains) where errors cascade and coverage guarantees decompose across stages. PASC provides joint coverage guarantees across all pipeline stages simultaneously. First conformal prediction framework with pipeline-aware joint coverage for multi-stage NLP/LLM pipelines.

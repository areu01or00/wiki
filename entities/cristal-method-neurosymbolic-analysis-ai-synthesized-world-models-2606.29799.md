# The CRISTAL Method: Neurosymbolic analysis from AI-synthesized world models

## Overview
CRISTAL (Coherent Reliable Intentional Synthesis of Truthful Analysis Logic) is a neurosymbolic framework for automating complex analysis workflows, with investment analysis as a primary use case. It addresses key challenges: high structural uncertainty, noisy and subjective data, tight attention budgets, and the need for justified, trustworthy outputs from LLM-based agents operating in high-stakes domains.

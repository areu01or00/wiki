# PERSA: Reinforcement Learning for Professor-Style Personalized Feedback with LLMs

## Overview
PERSA tackles the challenge of aligning LLM-generated feedback with specific instructor styles while preserving diagnostic accuracy. Using reinforcement learning, PERSA trains LLMs to generate professor-style personalized feedback that matches a given instructor's tone and pedagogical approach without sacrificing the correctness of educational guidance.

## Key Facts
- **Authors**: Ravi Ranjan, Utkarsh Grover, Xiaomin Lin, Agoritsa Polyzou
- **Year**: 2026
- **arXiv**: [2605.01123](https://arxiv.org/abs/2605.01123)

## Key Contributions
- RL framework for matching LLM feedback style to specific instructor personas without degrading feedback quality
- Addresses the style-vs-accuracy tradeoff in educational LLM feedback generation
- Demonstrates that style alignment is learnable via RL objective with appropriate reward shaping

# LLMs Uncertainty Quantification via Adaptive Conformal Semantic Entropy

## Overview
Proposes Adaptive Conformal Semantic Entropy (ACSE) for quantifying uncertainty in LLM outputs. Unlike token-level probability methods, ACSE operates on semantic equivalence classes — groups of outputs that convey the same meaning — and measures entropy over these classes to detect hallucination and overconfidence. Demonstrates superior calibration on factual QA and multi-hop reasoning tasks compared to standard semantic entropy approaches. First adaptive conformal method using semantic equivalence classes for LLM uncertainty quantification.

# HoneyTrap: Deceiving Large Language Model Attackers to Honeypot Traps with Resilient Multi-Agent Defense

## Overview
HoneyTrap proposes a deceptive defense framework where LLMs use collaborative honeypot traps to counter jailbreak attacks — instead of reactive refusal, the model proactively lures attackers into constrained sandboxes that expose their strategies. Multi-agent defenders coordinate to trap, analyze, and generalize from attack patterns, providing resilient defense that adapts to evolving jailbreak techniques.

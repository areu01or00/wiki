# SWE-Together: Evaluating Coding Agents in Interactive User Sessions

## Overview
SWE-Together introduces a multi-turn coding agent benchmark reconstructed from real user-agent coding sessions, addressing the gap between static agent benchmarks and interactive real-world coding assistance. The benchmark curates 109 repository-level tasks from 11,260 recorded sessions, enabling evaluation of agents that must adapt to clarifying goals, user constraints, and correction signals across multiple turns.

# LiveClawBench: Benchmarking LLM Agents on Complex, Real-World Assistant Tasks

## Overview
OpenClaw-style personal assistants extend LLM agents from isolated tool use to open-ended, stateful, personalized software environments. LiveClawBench introduces a benchmark evaluating assistant fidelity: benchmarks must be faithful both to real task distributions and to real software interfaces, addressing the fundamental evaluation fidelity problem for stateful agentic assistants.

## Key Facts
- **Authors**: Long, Xiang; Du, Li; Xu, Yilong + 11
- **Year**: 2026
- **arXiv**: [2604.13072](https://arxiv.org/abs/2604.13072)

## Key Contributions
- First benchmark for evaluating LLM assistants in stateful, open-ended software environments
- Defines fidelity as dual: task distribution fidelity + interface fidelity
- Addresses the evaluation gap between fixed benchmark tasks and real-world assistant deployment
- Enables reproducible evaluation of personal assistant agents across research teams

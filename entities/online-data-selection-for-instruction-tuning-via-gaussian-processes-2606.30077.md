# Online Data Selection for Instruction Tuning via Gaussian Processes

## Overview
This paper addresses the problem of data selection for instruction tuning of LLMs, proposing a Gaussian Process-based framework for online data selection that dynamically curates training data based on model learning signals.

## Key Facts
- **Authors**: Wang, Jun; Nguyen, Quoc Phong; Monteil, Julien + 1
- **Year**: 2026
- **arXiv**: [2606.30077](https://arxiv.org/abs/2606.30077)
- **Published**: 2026/06/29

## Key Contributions
- Gaussian Process-based online data selection framework for instruction tuning
- Dynamic curation of training data based on model learning signals
- Improved instruction-following performance vs random or static selection

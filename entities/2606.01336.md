# LongAttnComp: Cross-Family Context Compression for Long-Context Reasoning

## Overview
LongAttnComp addresses the critical bottleneck between context length and inference efficiency for 100k+ token inputs. The paper's key insight is that existing training-free attention-based context compression methods leave substantial accuracy gaps on demanding long-context tasks like code reasoning. LongAttnComp fine-tunes a lightweight cross-attention scoring layer and introduces token-level chunking with a token-budget top-p algorithm to compress context across different attention families (full attention vs. sliding window), achieving better accuracy-efficiency tradeoffs than prior training-free approaches while requiring minimal fine-tuning overhead.

## Key Contributions
- First cross-family context compression method bridging full-attention and sliding-window attention families for long-context reasoning
- Lightweight cross-attention scoring layer fine-tuning (minimal parameters) achieving substantial compression with preserved task accuracy
- Token-level chunking + token-budget top-p algorithm for adaptive compression budgeting
- Demonstrates significant improvements on code reasoning and long-document QA over training-free baselines
- Addresses prefill cost bottleneck that limits practical deployment of long-context LLMs

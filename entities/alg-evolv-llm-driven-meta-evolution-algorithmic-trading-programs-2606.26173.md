# AlgoEvolve: LLM-driven Meta-evolution of Algorithmic Trading Programs

## Overview
Recent work shows that Large Language Models (LLMs) can act as semantic mutation operators for the evolutionary discovery of programs and proofs. Most current applications focus on static coding benchmarks. AlgoEvolve extends this paradigm to algorithmic trading — a domain uniquely challenging because it is noisy, non-stationary, and highly discontinuous. The paper presents an LLM-driven meta-evolution system for discovering trading programs that adapt to market conditions.

## Key Facts
- **Authors**: Sharma, Dhruv; Shroff, Gautam
- **Year**: 2026
- **arXiv**: [2606.26173](https://arxiv.org/abs/2606.26173)

## Key Contributions
- First application of LLM-as-mutation-operator to non-stationary, noisy domain (algorithmic trading)
- Demonstrates LLM-driven evolutionary search discovers strategies that outperform static code baselines in changing market conditions
- Introduces semantic mutation operators that leverage LLM's world-knowledge for financial strategy discovery

# Agri-SAGE: Simulation-Grounded Multi-Agent LLM for Context-Aware Agricultural Advisory Generation

## Overview
Agri-SAGE addresses a fundamental tension in agricultural advisory systems: static agronomic guidelines provide consistent, evidence-based recommendations but are blind to in-season variability and dynamic uncertainties. The authors present a simulation-grounded multi-agent LLM system that generates context-aware agricultural advisories by integrating real-time data with agronomic simulation, combining the consistency of rule-based guidelines with the adaptability of LLM-driven recommendations.

## Key Facts
- **Authors**: Balasubramaniam, Vedant; Charan, Geetha; Patil, Manojkumar + 4

## Key Contributions
- Simulation-grounded multi-agent LLM architecture for dynamic agricultural advisory generation
- Integration of in-season variability and uncertainty quantification into LLM agent reasoning
- Demonstration of how multi-agent coordination can bridge rule-based agronomics and adaptive LLM recommendations

# Thinking Like a Clinician: A Cognitive AI Agent for Clinical Diagnosis

## Overview
This paper introduces DxChain, a chain-based clinical reasoning framework that transforms diagnostic workflow into an iterative process mirroring a clinician's cognitive trajectory through Memory Anchoring, Navigation, and Verification phases. It proposes three key innovations: Profile-Then-Plan paradigm, Medical Tree-of-Thoughts (Med-ToT) algorithm, and Dialectical Diagnostic Verification via adversarial debates. Achieves state-of-the-art on MIMIC-IV-Ext benchmarks.

## Key Contributions
- DxChain framework with three-phase cognitive trajectory: Memory Anchoring, Navigation, Verification
- Profile-Then-Plan paradigm to mitigate cold-start hallucinations via panoramic patient baseline
- Medical Tree-of-Thoughts (Med-ToT) algorithm for strategic lookahead planning
- Dialectical Diagnostic Verification using Angel-Devil adversarial debates for evidence conflict resolution
- State-of-the-art on MIMIC-IV-Ext Cardiac Disease and MIMIC-IV-Ext CDM benchmarks

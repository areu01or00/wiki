# Learning to Build the Environment: Self-Evolving Reasoning RL via Verifiable Environment Synthesis

## Overview
Self-improving language models typically generate problems or traces to imitate. This paper pursues a vision where the model constructs the environments that train it — reframing self-improvement from a data-generation loop into an environment-construction loop. Each artifact is a reusable executable object that samples instances, computes references, and scores responses.

## Key Facts
- **Authors**: Shi, Yucheng; Liang, Zhenwen; Panaganti, Kishan + 3
- **Year**: 2026

## Key Contributions
- Environment-construction loop for self-improving language models
- Reusable executable artifacts that sample, compute references, and score responses
- Zero-data reasoning RL via verifiable environment synthesis
- Single property determines whether the vision sustains improvement

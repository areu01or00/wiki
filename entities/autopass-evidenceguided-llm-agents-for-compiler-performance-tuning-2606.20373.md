# AutoPass: Evidence-Guided LLM Agents for Compiler Performance Tuning

## Overview
AutoPass uses LLM agents guided by compiler evidence (runtime profiles, microarchitectural feedback) to automatically tune compiler flags and optimization levels for improved LLM inference performance. The agent iteratively proposes, tests, and validates compiler configurations using actual benchmark runs, bridging the gap between LLM code generation and deployment-time performance optimization.

# SkillGrad: Optimizing Agent Skills Like Gradient Descent

## Overview
SkillGrad treats agent skill evolution as a gradient descent problem — instead of heuristic reflection-based skill refinement, the framework applies explicit optimization signals to update stored procedural knowledge. Skills (reusable procedural files) are optimized like model weights, enabling systematic improvement rather than ad-hoc revision.

## Key Facts
- **arXiv**: [2605.27760](https://arxiv.org/abs/2605.27760)

## Key Contributions
- Frames agent skill optimization as gradient descent on procedural knowledge files
- Addresses unreliable, incomplete, or outdated skills from third-party or self-generated sources
- Uses explicit optimization signals rather than heuristic reflection for skill evolution

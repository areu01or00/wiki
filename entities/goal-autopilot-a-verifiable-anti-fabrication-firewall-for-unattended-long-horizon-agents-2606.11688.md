# Goal-Autopilot: A Verifiable Anti-Fabrication Firewall for Unattended Long-Horizon Agents

## Overview
Long-horizon LLM agents confidently report success they never verified when run unattended. Goal-Autopilot treats honesty as a first-class metric distinct from capability, introducing a verifiable anti-fabrication firewall that bounds what an agent may claim at termination.

## Key Facts
- **arXiv**: [2606.11688](https://arxiv.org/abs/2606.11688)
- **Year**: 2026

## Key Contributions
- First framework treating agent honesty/bounded-claims as a distinct metric from capability
- Verifiable anti-fabrication firewall architecture for unattended autonomy
- Demonstrates honest termination guarantees for long-horizon agentic workflows

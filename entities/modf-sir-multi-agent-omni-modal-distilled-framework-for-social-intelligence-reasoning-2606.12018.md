# MODF-SIR: A Multi-agent Omni-modal Distilled Framework for Social Intelligence Reasoning

## Overview
MODF-SIR is a multi-agent collaborative framework built on a lightweight Multimodal Large Language Model for social intelligence reasoning. Knowledge distillation augments both training and inference phases, with multi-modal social intelligence data precisely localized and long-tail events extracted as formatted explicit text.

## Key Facts
- **Authors**: Shang Ma, Jisheng Dang, Wencan Zhang, Yifan Zhang, Bimei Wang, Hong Peng, Bin Hu, Qi Tian, Tat-Seng Chua
- **Year**: 2026
- **arXiv**: [2606.12018](https://arxiv.org/abs/2606.12018)

## Key Contributions
- Multi-agent MLLM framework for social intelligence reasoning
- Knowledge distillation at both training and inference phases
- Precise localization of multi-modal social intelligence data
- Long-tail event identification preventing information overshadowing

# Learning to Route LLMs from Implicit Cost-Performance Preferences via Meta-Learning

## Overview
Large language models (LLMs) present a trade-off between performance and cost, where more powerful models incur greater expense. LLM routing aims to mitigate expenses while maintaining performance by sending queries to the most suitable model. MetaRouter introduces a meta-learning-based approach that learns to route queries based on implicit cost-performance preferences, providing insights into advancing more intelligent LLM systems.

## Key Facts
- **Authors**: Zeng, Jiahao; Tang, Ming; Ding, Ningning
- **Year**: 2026
- **arXiv**: [2606.06178](https://arxiv.org/abs/2606.06178)
- **Date**: 2026-06-04

## Key Contributions
- First meta-learning-based LLM router that adapts to user cost-performance preferences
- Learns routing decisions from implicit preference signals rather than explicit feedback
- Demonstrates improved cost-performance trade-offs compared to fixed routing strategies

# Hierarchical Control in Multi-Agent Games: LLM-based Planning and RL Execution

## Overview
LLM-based high-level planning combined with RL for low-level execution in multi-agent games. The LLM plans at the strategic level while RL policies handle reactive execution, bridging symbolic planning with learned control in partially observable multi-agent environments.

## Key Facts
- **Authors**: Hösch, Jannik; Sestini, Alessandro; Fuchs, Florian + 5
- **Year**: 2026
- **arXiv**: [2606.20014](https://arxiv.org/abs/2606.20014)

## Key Contributions
- Hierarchical decomposition: LLM as strategic planner, RL as reactive controller
- Demonstrates pretrained LLM reasoning effectively orchestrates pretrained RL skills
- Scales to complex multi-agent partial-observability settings where pure RL fails
- Novel architecture bridging symbolic AI planning with reinforcement learning in multi-agent contexts

# H2HMem: A Multimodal Memory Benchmark for Agents in Human-Human Interaction

## Overview
H2HMem evaluates LLM agents deployed in human-human interaction settings (meeting assistants, clinical documentation). The benchmark tests memory recall, reasoning, and application across multimodal conversations, establishing that standard memory benchmarks fail to capture the demands of transcribing and reasoning over live human dialogue.

## Key Facts
- **Authors**: Zhu, Shiping; Yang, Yibo; Wang, Zhengyang + 3
- **Year**: 2026
- **arXiv**: [2606.09461](https://arxiv.org/abs/2606.09461)

## Key Contributions
- First benchmark specifically targeting LLM agents in human-human interaction contexts (meeting/clinical)
- Evaluates three modalities: memory recall, reasoning over stored dialogue, and applied task completion
- Demonstrates that existing memory benchmarks (designed for single-user or QA tasks) do not transfer to multi-party dialogue settings
- Provides unified evaluation framework for agent memory in realistic deployment scenarios

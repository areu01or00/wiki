# Reasoning Structure of Large Language Models

## Overview
Proposes that identical accuracy/token-count scores on reasoning benchmarks can hide fundamentally different reasoning structures. Introduces structural topology analysis of reasoning chains and defines a reasoning efficiency metric quantifying logical flow concentration. First work to treat reasoning structure as a measurable, topologically analyzable object.

## Key Contributions
- First framework to treat reasoning structure as a measurable topological object
- Novel reasoning efficiency metric based on logical flow concentration
- Demonstrates that final-answer accuracy metrics are insufficient for reasoning evaluation
- Enables quantitative comparison of reasoning strategies beyond token count

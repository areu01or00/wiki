# Agent-Native Immune System: Architecture, Taxonomy, and Engineering

## Overview
The Agent-Native Immune System (ANIS) is a biologically inspired, endogenous defense architecture embedded directly within an agent's cognitive reasoning loop — moving beyond external perimeter security and training-time alignment that leave agents vulnerable to runtime hijacking via memory poisoning, tool-chain manipulation, or multi-agent protocol attacks. The framework introduces a six-layer Immune Tower (L0-L5), a unified taxonomy of Agent Viruses and Agent Vaccines, the Harness Triad for continual immune learning, and a rigorous theoretical demarcation between model alignment (static constitutional foundation) and agent immunity (dynamic law enforcement during runtime).

## Key Contributions
- **Six-layer Immune Tower (L0-L5)**: Architectural framework with Barrier Immunity (L1) as a non-cognitive physical-and-logical isolation layer — the first immune tower architecture for LLM agents
- **Agent Virus and Agent Vaccine taxonomy**: Unified formal taxonomy distinguishing superficial non-parametric defenses from robust parametric vaccines — establishes the vocabulary for agent immune primitives
- **Harness Triad (Meta, Self, Auto)**: Self-monitoring, meta-cognitive automation backbone that drives Continual Immune Learning (CIL), enabling vaccines to dynamically adapt to novel threats
- **Alignment vs. immunity demarcation**: Theoretical framework establishing that alignment provides static constitutional values during training while ANIS serves as dynamic runtime law enforcement
- **Autoimmunity Rate metric**: Novel evaluation metric for the false-positive intervention rate in immune systems — opens the field of immune system evaluation for LLM agents

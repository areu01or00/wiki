# MultiUAV-Plat: An LLM-Oriented Platform, Benchmark and Framework for Multi-UAV Collaborative Task Planning

## Overview
Large language models provide a promising interface for high-level robotic task planning, but systematic evaluation of LLM-based multi-UAV collaboration has been lacking. MultiUAV-Plat bridges this gap with a lightweight, easy-to-use simulation platform specifically designed for LLM-agent-oriented multi-UAV collaborative task planning — capturing aerial-robotics constraints including partial observability, spatial coverage, UAV assignment, and multi-vehicle coordination that generic LLM benchmarks miss.

## Key Contributions
- Introduces MultiUAV-Plat: the first LLM-agent-oriented simulation platform for multi-UAV collaborative task planning
- Captures aerial-robotics constraints (partial observability, spatial coverage, multi-vehicle coordination) absent from generic LLM-agent benchmarks
- Provides a benchmark suite for evaluating LLM planning performance in multi-vehicle, spatially-distributed coordination scenarios
- First paper in the wiki to address LLM-based multi-UAV collaborative planning as a distinct domain from ground-robotics or single-agent LLM planning

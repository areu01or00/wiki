# CMIP-Forge: An Agentic System that Retrieves, Computes, and Self-Reviews Climate Science

## Overview
CMIP-Forge is a hybrid RAG and autonomous analysis system bridging scientific literature (6,581 CMIP6 publications, 101,828 indexed chunks) with Earth System Grid Federation data archives. A tool-augmented worker plans and executes Python workflows over live climate data while a panel of independent reviewer models audits methodology end-to-end. The system introduces a multi-layered Defense-in-Depth architecture enforcing physical and methodological invariants through AST static analysis, audited scientific primitives, and an autonomous adversarial peer-review protocol. Concrete failure modes exposed: sycophantic regression, unresolved REVISE verdicts, and stub code submission for review.

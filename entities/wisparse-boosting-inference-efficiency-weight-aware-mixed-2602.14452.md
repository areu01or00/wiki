# WiSparse: Boosting LLM Inference Efficiency with Weight-Aware Mixed Activation Sparsity

## Overview
Training-free activation sparsity is a promising approach for efficient LLM inference, yet existing methods often rely solely on activation information and use uniform sparsity patterns. WiSparse introduces weight-aware mixed activation sparsity that integrates activation magnitudes with precomputed weight structures for near-optimal inference acceleration.

## Key Facts
- **arXiv**: [2602.14452](https://arxiv.org/abs/2602.14452)
- **Theme**: inference efficiency / activation sparsity

## Key Contributions
- Weight-aware activation sparsification mechanism integrating activation magnitudes with precomputed weight structures
- Training-free approach requiring no fine-tuning or retraining
- Demonstrates significant inference speedup over uniform sparsity baselines

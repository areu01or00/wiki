# The Value of Variance: Mitigating Debate Collapse in Multi-Agent Systems via Uncertainty-Driven Policy Optimization

## Overview
Multi-agent LLM debate systems improve reasoning through iterative deliberation but suffer from "debate collapse" — a failure mode where agents converge on erroneous reasoning. This paper proposes uncertainty-driven policy optimization that uses variance in agent responses as a signal to detect and prevent collapse, allowing agents to continue deliberating when uncertainty is high rather than locking in wrong conclusions.

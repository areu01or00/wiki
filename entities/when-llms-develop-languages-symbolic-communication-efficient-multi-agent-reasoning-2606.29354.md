# When LLMs Develop Languages: Symbolic Communication for Efficient Multi-Agent Reasoning

## Overview
This paper reveals that Chain-of-Thought reasoning generates inefficient long natural-language rationales poorly aligned with machine reasoning. The authors propose Communicative Language Symbolism Routing (CLSR), a test-time framework where multiple LLM agents autonomously invent, evolve, and share compact Language Symbolism Frameworks (LSFs) — minimal symbolic representations of concepts that compress communication and accelerate multi-agent reasoning without sacrificing accuracy.

## Key Contributions
- Proposes Language Symbolism Frameworks (LSFs) — compact symbolic concept representations that LLM agents invent and share autonomously
- Shows CLSR reduces inter-agent communication tokens by 73% while maintaining reasoning accuracy on MATH and GAIA benchmarks
- Demonstrates emergent symbolic languages arising from multi-agent communication pressure rather than being hand-designed

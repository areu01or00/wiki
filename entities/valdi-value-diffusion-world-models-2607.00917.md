# Valdi: Value Diffusion World Models

## Overview
Valdi bridges the gap between diffusion models' expressive uncertainty modeling and the low-latency requirements of online model predictive control. It uses diffusion models for world dynamics prediction while enabling fast latent planning through a novel temporal diffusion inference scheme, making uncertain futures tractable for real-time control.

## Key Facts
- **Authors**: Lindenberg, Christopher; Chitta, Kashyap
- **Year**: 2026
- **arXiv**: [2607.00917](https://arxiv.org/abs/2607.00917)

## Key Contributions
- Latent diffusion world models enabling online Model Predictive Control (MPC)
- Bridges slow iterative diffusion inference with fast planning requirements
- Natural uncertainty quantification over multi-modal future trajectories
- First diffusion-based world model that achieves real-time control latency while preserving uncertainty awareness

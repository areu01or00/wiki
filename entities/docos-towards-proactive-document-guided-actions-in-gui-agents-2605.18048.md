# DocOS: Towards Proactive Document-Guided Actions in GUI Agents

## Overview
DocOS addresses a key limitation of GUI agents — their reliance on static parametric knowledge from pre-training — by enabling proactive document-guided action. The agent reads and reasons over documentation to handle long-tailed tasks that no training corpus could anticipate, allowing GUI agents to generalize to unseen tools and interfaces via document understanding.

## Key Facts
- **Authors**: Research team on GUI agents and document understanding
- **Year**: 2025
- **arXiv**: [2605.18048](https://arxiv.org/abs/2605.18048)

## Key Contributions
- Proactive document-guided framework enabling GUI agents to handle long-tailed tasks via documentation reasoning
- Addresses the generalization gap from static parametric knowledge in existing GUI agents
- Enables zero-shot tool and interface adaptation through document understanding rather than retraining

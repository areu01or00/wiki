# HoloAgent-0: A Unified Embodied Agent Framework with 3D Spatial Memory

## Overview
HoloAgent-0 extends the LLM agent execution loop (reason, tool use, feedback, revision) from digital environments to physical robots by managing 3D spatial memory representations. It addresses the challenges of continuous, embodiment-dependent, uncertain physical execution by maintaining spatial state that is shared across perception, planning, and action — enabling agents to reason about object locations, affordances, and spatial constraints.

## Key Facts
- **Authors**: Anonymous (arXiv submitter, 2026-06)

## Key Contributions
- Unifies embodied AI capabilities (manipulation, spatial understanding, navigation, humanoid control) under a single 3D spatial memory framework
- Addresses embodiment-dependent uncertainty via continuous spatial representations rather than discrete symbolic states
- Extends the agentic execution loop (reason → tool → feedback → revise) to physical environments with safety constraints
- Provides a shared spatial memory across perception, planning, and action modules

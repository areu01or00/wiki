# Title:Just-In-Time Reinforcement Learning: Continual Learning in LLM Agents Without Gradient Updates

## Overview
LLM agents struggle with continual adaptation due to frozen weights after deployment. Conventional RL solutions are computationally prohibitive and risk catastrophic forgetting. JitRL introduces a training-free framework enabling test-time policy optimization without any gradient updates, maintaining a dynamic non-parametric policy that adapts at inference time.

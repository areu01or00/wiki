# Can LLMs Be CEOs? Benchmarking Strategic Resource Reallocation with Multi-Role Agent Simulation

## Overview
This paper introduces the Multi-Role Executive Benchmark (MREB), testing LLMs' ability to play multiple executive roles simultaneously and make strategic resource allocation decisions under conflicting stakeholder priorities — finding a 34% performance drop when integrating multi-role inputs versus single-role tasks.

## Key Facts
- **arXiv**: [2606.17459](https://arxiv.org/abs/2606.17459)
- **Year**: 2026
- **Theme**: LLM executive strategic benchmarking

## Key Contributions
- Introduces Multi-Role Executive Benchmark (MREB): simulation where LLMs play CEO/CFO/CTO/COO simultaneously
- Evaluates 12 frontier LLMs; find 34% average performance drop on multi-role integration vs single-role tasks
- Identifies root causes: models fail to weight competing priorities and exhibit positional bias toward first recommendation
- First benchmark to systematically test executive decision-making under budget/time constraints in LLMs
- Reveals gap between benchmark saturation and genuine executive capability in frontier models

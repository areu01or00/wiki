# Reducing Conversational Escalation in Large Language Model Dialogue with Nonviolent Communication Constraints

## Overview
This paper investigates whether Large Language Models can be guided toward de-escalating dialogue behavior through lightweight prompt-level constraints derived from Nonviolent Communication (NVC) principles. The work addresses a critical gap in LLM safety research: while explicit harms have been extensively studied, conversational behaviors that unintentionally escalate conflict remain underexplored.

# MUSE: A Unified Agentic Harness for MLLMs

## Overview
MUSE introduces a multimodal unified structured execution harness that wraps any off-the-shelf MLLM to improve task execution without retraining. Addresses the gap where MLLMs fail on tasks humans solve effortlessly (grid maze navigation, puzzle piece selection) by improving the execution scaffold rather than the model weights. Uses structured execution to elicit latent capabilities from frozen MLLMs.

## Key Contributions
- Unified agentic harness for MLLMs: wraps frozen models with structured execution scaffold
- Demonstrates capability elicitation without retraining — purely via execution engineering
- Addresses tasks humans solve effortlessly but MLLMs fail (grid navigation, visual puzzle selection)
- Architecture-agnostic: works with any off-the-shelf MLLM
- Introduces structured execution as complement to model improvement

# SocialMemBench: Are AI Memory Systems Ready for Social Group Settings?

## Overview
SocialMemBench identifies a critical gap: existing memory benchmarks evaluate dyadic or workplace dialogue, but none targets multi-party social groups where memory must anchor to social context, group dynamics, and relationship history. The paper demonstrates that current AI memory systems fail characteristically in social group settings — a key gap for group-acting agents embedded in chat platforms and proactive personal assistants.

## Key Facts
- **Authors**: Owolabi, Olukunle
- **Year**: 2025
- **arXiv**: [2605.17789](https://arxiv.org/abs/2605.17789)

## Key Contributions
- First benchmark targeting multi-party social group memory — distinct from dyadic/workplace benchmarks
- Identifies characteristic failure modes of memory systems in social group contexts
- Proposes social-context-anchored memory as a new evaluation dimension
- Gap-filling benchmark for group-acting agents in chat platforms and social assistants

# ConcoLixir: Reactive LLM Discovery Oracles for Python Concolic Testing

## Overview
Concolic testing combines concrete execution with symbolic constraint solving, but Python programs expose limits when symbolic variables downgrade to concrete values. ConcoLixir introduces reactive LLM discovery oracles that augment concolic testing by using LLMs to propose novel test inputs at stuck exploration states. The reactive oracle pattern (LLM-as-exploration-catalyst) is orthogonal to prior tool-use and benchmark papers by treating the LLM not as an agent executing a plan but as an oracle providing discovery hints at symbolic impasse points.

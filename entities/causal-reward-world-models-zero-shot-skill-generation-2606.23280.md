# Causal Reward World Models: Zero-shot Reward Design for Automated Skill Generation

## Overview
Automated Reward Design (ARD) replaces manual reward engineering in RL with language-driven reward synthesis. Existing LLM-based approaches are correlation-driven. This paper introduces Causal Reward World Models — using structural causal models for zero-shot reward design that captures true causal relationships rather than correlations.

## Key Contributions
- Causal reward models that replace correlation-driven LLM reward synthesis
- Zero-shot automated RL skill generation via causal world models
- Structural causal models for reward function synthesis

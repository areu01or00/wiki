# RocketSmith: Agentic Additive Manufacturing of High-Powered Rockets

## Overview
RocketSmith is an agentic system which intelligently automates the DFAM (Design for Additive Manufacturing) process for the development of high-powered rockets suitable for launch. The system utilizes a large language model to orchestrate the execution of software tools to validate design characteristics such as flight stability and generate parametric design components for rocket assembly. A collection of subagents and skills enable optimization workflows of flight parameters via iteration in both zero-shot and human-in-the-loop workflows. Four distinct high-power rockets with various motor and assembly configurations were developed using the unique design capabilities of additive manufacturing. Assembly components were fabricated using FDM printers, manually evaluated for flight readiness, and flight tested at a launch event. All rockets achieved stable launch and two of four were successfully recovered in reflyable condition. Altimeter data validated that rockets achieved an altitude 80% of the expected apogee predicted by the agentic system, establishing consistency between simulation and experimentation. First system in the wiki combining LLM agentic orchestration with DFAM rocket engineering.

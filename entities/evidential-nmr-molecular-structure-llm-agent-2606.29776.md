# Towards Generalizable and Evidential NMR-Based Molecular Structure Elucidation via Large Language Model Agent

## Overview
NMRAgent addresses the trade-off between database retrieval (cannot identify novel scaffolds) and black-box de novo models (lack atom-level interpretability) in NMR spectroscopy interpretation. It introduces an evidential reasoning agent that combines the gold-standard NMR spectroscopy with LLM-powered autonomous reasoning for molecular structure elucidation — providing both novel-scaffold identification and rigorous scientific validation.

## Key Facts
- **Authors**: Zheng Fang, Chen Yang, Yusen Tan, Yunpeng Zhao, Fanjie Xu, Hongxin Xiang, Hanyu Sun, Hanyu Gao, Xiaojian Wang, Wenjie Du, Yuqiang Li, Jun Xia
- **Year**: 2026
- **arXiv**: [2606.29776](https://arxiv.org/abs/2606.29776)

## Key Contributions
- First LLM agent framework for NMR molecular structure elucidation combining database retrieval depth with de novo generation breadth
- Evidential reasoning layer providing atom-level interpretability required for scientific validation
- Addresses the novel-scaffold identification problem that pure retrieval methods cannot solve
- Demonstrates generalizability to molecular structures outside training distribution

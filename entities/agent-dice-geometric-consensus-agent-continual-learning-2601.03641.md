# Title:Agent-Dice: Disentangling Knowledge Updates via Geometric Consensus for Agent Continual Learning

## Overview
LLM-based agents face the stability-plasticity dilemma when continually learning new tasks without catastrophic forgetting. This paper argues the dilemma arises from failure to distinguish common knowledge shared across tasks from conflicting task-specific knowledge. Agent-Dice uses geometric consensus to disentangle knowledge updates, enabling agents to update knowledge while preserving shared foundations.

# Event-B Agent: Towards LLM Agent for Formal Model Synthesis and Repair

## Overview
Event-B Agent bridges LLMs with formal methods by using Event-B as a structured specification language for LLM agents to synthesize and repair formal models. The agent iteratively proposes formalizations, attempts proof, and repairs failed proof obligations — bringing correct-by-construction software engineering practices into the LLM agent loop.

## Key Facts
- **Authors**: A team exploring LLM + formal synthesis
- **Year**: 2025
- **arXiv**: [2605.17475](https://arxiv.org/abs/2605.17475)

## Key Contributions
- First LLM agent framework that uses Event-B as the formal specification substrate for iterative synthesis and proof repair
- Demonstrates that LLMs can handle the feedback loop between failed proof obligations and model revision
- Bridges software engineering formal methods (Event-B) with LLM agentic reasoning

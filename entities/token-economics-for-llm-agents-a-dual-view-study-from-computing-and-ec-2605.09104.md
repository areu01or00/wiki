# Token Economics for LLM Agents: A Dual-View Study from Computing and Economics

## Overview
This paper treats tokens as the core economic primitives of Agentic AI — introducing a dual-view framework (computing-cost view + economic-value view) for understanding token consumption in LLM agent workflows. It analyzes how iterative agent loops (reasoning, tool use, self-correction) drive exponential token growth and proposes adaptive allocation strategies.

## Key Facts
- **Authors**: Chen, Yuxi; Chen, Junming; He, Chenyu + 9
- **Year**: 2026
- **arXiv**: [2605.09104](https://arxiv.org/abs/2605.09104)

## Key Contributions
- Dual-view token economics model: computing-cost perspective + economic-value perspective
- First systematic analysis of token as economic primitive in agentic AI
- Identification of exponential token consumption patterns in iterative agent loops
- Adaptive token allocation framework for agent cost-efficiency

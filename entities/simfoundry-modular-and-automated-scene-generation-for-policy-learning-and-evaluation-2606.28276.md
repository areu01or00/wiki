# SimFoundry: Modular and Automated Scene Generation for Policy Learning and Evaluation

## Overview
SimFoundry introduces a modular system for zero-shot real-to-sim scene construction from video, generating sim-ready digital twins and supporting object, scene, and task editing for robot policy learning. Policies trained on SimFoundry data transfer zero-shot to challenging real-world scenes, enabling scalable embodied agent evaluation without costly real-world data collection.

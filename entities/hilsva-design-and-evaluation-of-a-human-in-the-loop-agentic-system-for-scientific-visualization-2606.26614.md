# HiLSVA: Design and Evaluation of a Human-in-the-Loop Agentic System for Scientific Visualization

## Overview
HiLSVA addresses a fundamental tension in LLM-based scientific visualization systems: prior agentic systems prioritize autonomy over human analytical control, limiting transparency and oversight. The authors present a human-in-the-loop agentic architecture for scientific visualization (SciVis) that maintains agent capabilities while preserving human agency and analytical control through structured intervention points.

## Key Facts
- **Authors**: Ai, Kuangshi; Do, Patrick Phuoc; Wang, Chaoli
- **Year**: 2026
- **arXiv**: [2606.26614](https://arxiv.org/abs/2606.26614)

## Key Contributions
- Novel human-in-the-loop agentic architecture for scientific visualization workflows
- Structured intervention mechanisms that preserve autonomy while enabling human oversight
- Comparative evaluation showing HiLSVA outperforms fully-autonomous agents on transparency metrics without sacrificing task performance

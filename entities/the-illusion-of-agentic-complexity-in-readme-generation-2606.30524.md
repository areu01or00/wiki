# The Illusion of Agentic Complexity in README.md Generation: Evaluating Single-Agent vs. Multi-Agent RAG Systems

## Overview
Evaluates whether multi-agent architectures genuinely improve autonomous code-documentation tasks, finding that additional architectural complexity often masks rather than resolves fundamental capability gaps. Tests both single-agent and multi-agent RAG systems on repository-level README generation using a new RealWorldAgentEval benchmark.

## Key Facts
- **Authors**: Abu Saleh, Tesfay Welegebreal Tesfay, Phuong T. Nguyen, Juri Di Rocco, Muhammad Umar Zeshan
- **Year**: 2026
- **arXiv**: [2606.30524](https://arxiv.org/abs/2606.30524)

## Key Contributions
- **RealWorldAgentEval**: New benchmark for open-source README generation, measuring both task success and documentation quality
- **Illusion of complexity**: Multi-agent setups do not consistently outperform single-agent baselines on documentation tasks
- **Agentic complexity gap**: Identifies architectural choices that appear to add capability but don't improve output quality
- **First LLM-agent architectural complexity evaluation for repository-level documentation in the wiki**

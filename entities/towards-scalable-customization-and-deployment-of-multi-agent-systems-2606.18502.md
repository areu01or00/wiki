# Towards Scalable Customization and Deployment of Multi-Agent Systems for Enterprise Applications

## Overview
This paper proposes a unified framework for customization and efficient deployment of multi-agent LLM systems in real-world enterprise settings. The framework combines continual pretraining, supervised fine-tuning, and preference optimization (Agentic Model Customization) with speculative decoding and FP8 quantization (Inference Optimization) to enable rapid domain adaptation with 4.48x throughput improvements.

- Unified framework combining model customization (continual pretraining + SFT + preference optimization) with inference optimization (speculative decoding + FP8 quantization)

- First framework systematically addressing both domain adaptation AND inference efficiency for multi-agent LLM systems
- Works across enterprise workloads with robust generalization

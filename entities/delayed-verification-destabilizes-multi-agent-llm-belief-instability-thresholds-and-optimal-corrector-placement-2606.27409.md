# Delayed Verification Destabilizes Multi-Agent LLM Belief: Instability Thresholds and Optimal Corrector Placement

## Overview
Models how delayed verification in multi-agent LLM systems causes false claims to propagate through agent networks before correction arrives. Using spectral decomposition via the grounded Laplacian, derives a closed-form stability threshold for verification dose — showing that correction that is too strong or too delayed flips consensus into oscillation. Most unstable: when communication and verification delays coincide. Provides principled corrector placement guidance for multi-agent LLM deployment.

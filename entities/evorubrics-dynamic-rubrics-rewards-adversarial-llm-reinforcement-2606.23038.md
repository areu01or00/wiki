# EvoRubrics: Dynamic Rubrics as Rewards via Adversarial Co-Evolution for LLM Reinforcement Learning

## Overview
CPE (Causal Perturbative Elicitation) is an unsupervised method for discovering diverse, generalizable perturbations of LLM internals that surface hidden behavioral modes. These perturbations can reshape model behavior and systematically evaluate potential risks — providing a mechanistic interpretability lens into latent LLM capabilities.

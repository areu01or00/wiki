## Picks

1. **Learning What to Forget: Improving LLM Unlearning via Learned Token-Level Importance** (2606.06320) — Introduces Alternating Token-Weighted Unlearning (ATWU) with joint optimization over model parameters and token weights; characterizes token forget-specificity through retain-conflict interaction; achieves SOTA forget-retain trade-offs on TOFU and RWKU; orthogonal to KnowledgeSmith (model-editing-based unlearning) and gradient-ascent baselines.
2. **Selective Forgetting for Large Reasoning Models** (2604.03571) — RAG-based CoT trace analysis to identify forget-relevant segments in LRMs; feature replacement unlearning loss suppresses forgotten content while reinforcing valid replacements; preserves general reasoning while achieving precise LRM-specific unlearning; orthogonal to ATWU's retain-conflict oracle approach (different mechanism for different model class).
3. **TMAS: Scaling Test-Time Compute via Multi-Agent Synergy** (2605.10344) — Organizes inference as collaborative process among specialized agents with hierarchical memories (experience bank + guideline bank); hybrid reward RL scheme jointly preserves reasoning capability, experience utilization, and exploration; stronger iterative scaling than existing test-time scaling baselines; orthogonal to single-model test-time compute papers (When More Thinking Hurts) and self-training collapse papers (Self-Improvement Can Self-Regress).
